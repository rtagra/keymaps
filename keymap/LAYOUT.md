## Raise Layer (_RAISE)

Numbers, symbols, and firmware reset.

```
┌──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┐
│  `   │  1   │  2   │  3   │  4   │  5   │  6   │  7   │  8   │  9   │  0   │ Del  │
├──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│      │      │      │      │      │      │      │  -   │  =   │  [   │  ]   │  \   │
├──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│      │      │      │      │      │      │      │      │      │      │      │      │
├──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│Reset │      │      │      │      │      │      │ #### │      │      │      │      │
└──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

**Legend:**
- `####` = Active layer key (the key you're holding to access this layer)
- Empty cells = Transparent (passes through to lower layer)

## Key Features

- **Momentary Layer Access**: Hold Lower, FN, or Raise to access those layers
- **Media Controls**: On FN layer (Mute, Volume, Play/Pause)
- **Navigation**: Arrow keys on base layer, extended navigation (Home/End/PgUp/PgDn) on FN layer
- **Symbols**: Quick access to symbols on Lower and Raise layers
- **Firmware Reset**: QK_BOOT on Raise layer (bottom-left key) for flashing firmware
